# Amanah Law Firm - Website Profil Perusahaan

Website profil perusahaan profesional untuk **Amanah Law Firm** yang dibangun menggunakan framework modern **Astro** dan dideploy secara live melalui platform **Vercel**.

## ⚖️ Fitur Utama
- **Beranda:** Halaman utama dengan desain modern dan profesional.
- **Tentang Kami:** Informasi mengenai visi, misi, dan latar belakang firma hukum.
- **Bidang Hukum:** Daftar layanan hukum dan spesialisasi yang ditangani.
- **Tim Pengacara:** Profil para pengacara profesional dan berintegritas.
- **Kontak & Konsultasi:** Akses cepat untuk menghubungi layanan konsultasi gratis.

## 🚀 Teknologi yang Digunakan
- **Framework:** Astro
- **Deployment:** Vercel
